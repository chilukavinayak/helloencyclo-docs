# HelloEncyclo Documentation Vault

Welcome to your Obsidian documentation vault!

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `00 Inbox` | Quick capture of new notes and ideas |
| `01 Notes` | Permanent notes and knowledge base |
| `02 Projects` | Project-specific documentation |
| `03 Resources` | Reference materials, links, articles |
| `04 Archive` | Completed or inactive notes |
| `Templates` | Note templates for consistency |
| `Attachments` | Images, PDFs, and other files |

## Getting Started

1. Open this folder as a vault in Obsidian
2. Use `00 Inbox` for quick captures
3. Process inbox items into `01 Notes` regularly
4. Link notes using `[[WikiLinks]]`
5. Commit changes with Git to track your documentation history

## Git Workflow

```bash
# Stage and commit your documentation changes
git add .
git commit -m "docs: your message here"

# View history
git log --oneline
```
